Built a RAG-based chatbot capable of answering questions from PDFs using text
extraction, smart chunking, and vector-style retrieval.
---------------------------------------------------------------------------------
Integrated AWS Bedrock (Sonnet 3.5) for LLM responses with a fallback mock
mode for local development.
--------------------------------------------------------------------------------
Also Developed a complete backend using FastAPI, including JWT authentication,
session-based conversation memory, and modular API routing.
---------------------------------------------------------------------------------
Designed PDF ingestion pipeline, embedding generation, and stored processed
chunks using SQLModel + SQLite.
--------------------------------------------------------------------------------
Implemented clean architecture with environment-based configuration using
.env, ensuring scalable and production-ready code.
